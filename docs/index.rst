==================
carbon-client-node
==================

``carbon-client-node`` is the Node.js client for Carbon.io services. It's an extension of the :ref:`carbon-client-js <carbon-client-js>`
with more Node.js abilities, mainly allowing synchronized style calls.

------------
Installation
------------

.. code::


    <path-to-your-app>/
      package.json


Your ``package.json`` should include ``carbon-client-node``

.. code:: javascript


    {

      "dependencies": {
          "@carbon-io/carbon-client-node": "0.3.x"
      }

    }


Then install the package dependencies like this:

.. code::


    % cd <path-to-your-app>
    % npm install .
